# MAC Address Changer

A simple dark-mode GUI tool to generate, apply, and reset MAC addresses.

## Features
- Generate random MAC addresses.
- Apply MAC changes (requires admin).
- Reset to default MAC.
- Check current MAC.
- Copy to clipboard.
